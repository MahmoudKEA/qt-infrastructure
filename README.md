# Qt Infrastructure
#### This infrastructure is designed to build qt applications more organized, clean, flexible and scalable 
- Designed by: Mahmoud Khalid

## Requirements for Python 3
    pip install PyQt6

## Infrastructure Scheme
    launcher.py
        | packages
            | events
                | eventcaller.py
                | eventform.py
                | eventsignals.py
            | header.py
            | utility.py
        | ui
            | styles
                | head.py
            | images
                | icons
                | head.py
            | fonts
                | head.py
            | languages
            | main.py
        | src
            | main.py
        | settings
            | config.json
        | logs
